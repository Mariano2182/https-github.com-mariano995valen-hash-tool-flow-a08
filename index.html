    Modelo de datos BIM

    No guardamos geometría suelta, guardamos decisiones de ingeniería.

    Proyecto

    Raíz del sistema: project_id, nombre, cliente, estado, versión_actual.

    Relaciones: 1 Proyecto → N Naves · 1 Proyecto → N Versiones.

    Nave

    nave_id, ancho, largo, altura_libre, modulación, tipo_nave.

    Relaciones: Pórticos, Correas, Arriostramientos.

    Pórtico

    portico_id, posición_X, tipo.

    Relaciones: 2 Columnas + 1 Viga.

    Elemento estructural

    elemento_id, tipo, perfil_id, material, longitud, orientación.

    Relación: 1 Elemento → N Uniones.

    Flujo completo BIM – Cliente → Ingeniería → Fabricación

    Una sola fuente de verdad: modelo BIM paramétrico.

    Ingeniería primero, tecnología al servicio del modelo.

    • React + TypeScript
    • Vite
    • Three.js + WebGL

    • Node.js (NestJS) o Python (FastAPI)

    • Orquestación de flujos
